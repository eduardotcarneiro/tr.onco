Descrição do Projeto

O trabalho realizado (intitulado TR.ONCO) teve como objetivo analisar e prototipar uma solução para auxiliar a oncologia pediátrica. O software se assemelha a um prontuário eletrônico. O software fim consistiria de uma base de dados que lista os usuários que estão em tratamento, e nele possuímos os dados dos pacientes, histórico de diagnósticos, e sugestão de diagnósticos com base em características apresentadas pelo usuário. Desta maneira, o aplicativo ajudaria o profissional a selecionar o problema, através dos sintomas observados.
